# Phonebook Frontend

## Configure backend URL

Create `.env` and set:

```bash
VITE_API_BASE_URL=https://your-backend-url.onrender.com
```

For local development, you can use:

```bash
VITE_API_BASE_URL=http://localhost:3001
```

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
